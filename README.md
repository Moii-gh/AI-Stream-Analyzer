**AI Stream Analyzer** — это мощное веб-приложение для анализа аудио и видео потоков в реальном времени с использованием передовых возможностей искусственного интеллекта (Google Gemini 3.1 Flash).

🔗 **[Официальный репозиторий на GitHub](https://github.com/Moii-gh/AI-Stream-Analyzer)**

Приложение захватывает ваш экран или веб-камеру, распознает речь и мгновенно предоставляет структурированную аналитику, ревью кода, критику и саммари происходящего.

---

## ✨ Ключевые возможности

*   📹 **Захват экрана и камеры:** Легко переключайтесь между трансляцией экрана (для презентаций или лайвкодинга) и веб-камерой.
*   🎤 **Распознавание речи (Speech-to-Text):** Транскрибация вашего голоса в текст в реальном времени.
*   🧠 **Глубокий AI-анализ:** Каждые несколько секунд ИИ анализирует кадры видео и вашу речь, выдавая:
    *   📝 Структурированное саммари.
    *   💡 Ключевые идеи.
    *   💻 Ревью кода (если на экране есть код, ИИ предложит оптимизации).
    *   🎯 Конструктивную критику (улучшение UI/UX, подачи материала).
*   💬 **Интерактивный чат:** Задавайте вопросы ИИ прямо во время трансляции, опираясь на текущий контекст.
*   📊 **Визуализация аудио:** Красивый осциллограф, реагирующий на ваш голос.

---

## 🛠 Технологический стек

*   **Frontend:** React 19, TypeScript, Vite
*   **Стилизация:** Tailwind CSS v4, Lucide React (иконки)
*   **Анимации:** Motion (Framer Motion)
*   **ИИ & LLM:** `@google/genai` (Gemini 3.1 Flash Preview)
*   **Markdown:** `react-markdown` для красивого рендеринга ответов ИИ

## 🚀 Как запустить локально

1. **Клонируйте репозиторий и перейдите в папку:**
   ```bash
   git clone <url-репозитория>
   cd ai-stream-analyzer
   ```

2. **Установите зависимости:**
   ```bash
   npm install
   ```

3. **Настройте переменные окружения:**
   Создайте файл `.env` в корне проекта (или скопируйте из `.env.example`) и добавьте ваш API ключ Gemini:
   ```env
   GEMINI_API_KEY="ваш_ключ_gemini_здесь"
   ```

4. **Запустите сервер разработки:**
   ```bash
   npm run dev
   ```

5. Откройте `http://localhost:3000` в вашем браузере.

## 💡 Сценарии использования

*   **Лайвкодинг и стримы:** Получайте мгновенную обратную связь по вашему коду и архитектуре прямо во время написания.
*   **Подготовка к презентациям:** Репетируйте выступления и получайте критику по подаче и визуалу слайдов.
*   **Обучение:** Анализируйте обучающие видео или лекции, автоматически генерируя конспекты и выделяя главные мысли.

---

## 🔒 Приватность и разрешения

Приложению требуются следующие разрешения браузера для корректной работы:
*   `camera` — для захвата видео с веб-камеры.
*   `microphone` — для транскрибации речи и визуализации аудио.
*   `display-capture` — для захвата экрана.

*Все данные обрабатываются локально в браузере и отправляются в Google Gemini API только для генерации аналитики.*
